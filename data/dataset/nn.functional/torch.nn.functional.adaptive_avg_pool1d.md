torch.nn.functional.adaptive_avg_pool1d 
====================================================================================================================

torch.nn.functional. adaptive_avg_pool1d ( *input*  , *output_size* ) → [Tensor](../tensors.html#torch.Tensor "torch.Tensor") 
:   Applies a 1D adaptive average pooling over an input signal composed of
several input planes. 

See [`AdaptiveAvgPool1d`](torch.nn.AdaptiveAvgPool1d.html#torch.nn.AdaptiveAvgPool1d "torch.nn.AdaptiveAvgPool1d")  for details and output shape. 

Parameters
: **output_size** – the target output size (single integer)
